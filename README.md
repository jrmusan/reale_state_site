## How to Use

1. **View the webpage**: Simply open `index.html` in any web browser
2. **Customize your information**:
   - Replace the agent photo by updating `agent-photo.svg` with your own image (JPEG, PNG, or SVG)
   - Edit `index.html` to update:
     - Your name and tagline
     - About Me description
     - Phone number
     - Email address
     - Office address
     - Office hours
3. **Deploy**: Upload all files to any web hosting service

## Files

- `index.html` - Main webpage
- `style.css` - Styling and responsive design
- `agent-photo.svg` - Placeholder agent photo (replace with your own)
